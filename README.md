# ED_Lista04.05
## Algoritmos de Ordenação

Este desafio consiste em criar uma aplicação, em Java, que receba duas bibliotecas com algoritmos de ordenação e testa-los com os seguintes dados:
```
                                        Vetor1: {74,20,74,87,81,16,25,99,44,58}
                                        Vetor2: {44,43,42,41,40,39,38}
                                        Vetor3: {101,102,103,125,124,123,104,105,106,122}
```

```
Requisitos do projeto:

1. Arquitetura MVC;
2. Os vetores devem ser declarados no Main;
3. As bibliotecas devem ser acessadas por uma classe no package controller.
```
